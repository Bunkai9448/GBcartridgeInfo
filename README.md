# GBcartridgeInfo

Used to quickly visualize (and/or use) the key values in a Gameboy cartridge

**Documentation used**
- https://gbdev.io/pandocs/The_Cartridge_Header.html


**Prerequisites**
- Python version used: 3.7.3
- A gb ROM file
